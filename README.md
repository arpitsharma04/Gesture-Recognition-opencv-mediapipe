# Hand Gesture Recognition System

This project detects hand gestures in real-time using OpenCV and MediaPipe.

## Features
- Detects gestures like:
  - OK 👍
  - Disapproval 👎
  - Hello ✋
  - Stop ✊
  - Sorry ✌
  - Love ❤️
  - Thank You 🙌

## Tech Used
- Python
- OpenCV
- MediaPipe
- NumPy

## How to Run
1. Install dependencies:
pip install -r requirements.txt

2. Run:
python gesture_detector.py

## Controls
- Press S → Start
- Press Q → Quit

## Note
Use Python 3.10 or 3.11 (MediaPipe not supported in 3.13)
